# Exercice 2 : La vie d'Ada 2.0 (HTML5 / CSS3)

## Description

Cet exercice consiste à recréer une page web simple en HTML et CSS présentant la vie d’Ada Lovelace, à partir d'un visuel fourni.

L’objectif est de pratiquer :

- La structure HTML de base
- La mise en forme avec CSS commande de base
- Accrocher une image, et du style à la page

## Technologies utilisées

- HTML5
- CSS3

## Structure du projet

```text

exercice-02/
├── README.md
├── index.html
├── style.css
└── img/
    └── img-ada.jpeg
    └── img-exercice02.png

```

## Contenu de la page :

La page contient :

- Un titre principal
- Un texte de présentation d’Ada Lovelace
- Une image portrait
- Une liste de ses contributions majeures

## Consignes graphiques

- Couleurs : rose et hotpink
- Police : Comic Sans MS
- Mise en page centrée
- Image arrondie avec bordure couleur: hot pink

## Comment lancer le projet

1. Télécharger tous les fichiers du projet
2. Ouvrir le fichier index.html dans un navigateur web

## Auteur·e

Projet réalisé dans le cadre d’un exercice HTML/CSS pour une formation de Dev FUll Stack à Ada Tech School de Nantes.
