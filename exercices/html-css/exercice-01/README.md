# Exercice 1 : La vie d'Ada (HTML5)

## Description

Cet exercice consiste à recréer une page web simple en HTML présentant la vie d’Ada Lovelace, à partir d'un visuel fourni.

L’objectif est de pratiquer :

- La structure HTML de base
- Ajouter une image
- Mettre en forme sans feuille de style

## Technologies utilisées

- HTML5

## Structure du projet

```text

exercice-01/
├── README.md
├── index.html
├── style.css
└── img/
    └── img-ada.jpeg
    └── img-exercice01.png

```

## Contenu de la page :

La page contient :

- Un titre principal
- Un texte de présentation d’Ada Lovelace
- Une image portrait
- Une liste de ses contributions majeures

## Consignes graphiques

- Couleur et mise en page : par défaut
- Mettre en évidence un mot
- Redimensionner une image
- Créer une liste à puces

## Comment lancer le projet

1. Télécharger tous les fichiers du projet
2. Ouvrir le fichier index.html dans un navigateur web

## Auteur·e

Projet réalisé dans le cadre d’un exercice HTML/CSS pour une formation de Dev FUll Stack à Ada Tech School de Nantes.
